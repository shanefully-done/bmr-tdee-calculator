# Technology Stack

## 1. Core Frameworks & Libraries

- **Next.js 15**: The primary React framework for building the application.
- **React 19**: The core UI library.
- **TypeScript**: For static typing and improved code quality.

## 2. UI & Styling

- **shadcn/ui**: A collection of reusable UI components built on Radix UI and Tailwind CSS.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Radix UI**: Provides accessible, unstyled UI primitives that power `shadcn/ui`.
- **Lucide React**: A library of simply designed icons.

## 3. Forms & State Management

- **React Hook Form**: For managing form state and validation.
- **Zod**: A TypeScript-first schema declaration and validation library, used with React Hook Form.

## 4. Development & Tooling

- **Bun**: Used as the package manager, as indicated by `bun.lockb`.
- **ESLint**: For code linting and maintaining code quality.
- **Prettier**: For code formatting.
- **Turbopack**: The Rust-based successor to Webpack, used for development and builds via `next dev --turbopack`.
