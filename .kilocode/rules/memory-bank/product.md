# Product: BMR & TDEE Calculator

## 1. Vision

A simple, fast, and responsive web application to help users calculate their Basal Metabolic Rate (BMR) and Total Daily Energy Expenditure (TDEE). The primary goal is to provide an accurate and easy-to-use tool for health and fitness tracking.

## 2. Core Problem

Users need a straightforward way to determine their daily caloric needs without complex calculations or navigating ad-heavy websites. This app solves that by offering a clean, focused, and immediate calculator.

## 3. User Experience

- **Simplicity:** The user journey is designed to be as simple as possible. The initial view is focused solely on BMR calculation.
- **Progressive Disclosure:** TDEE calculation is offered as a secondary step after the BMR is calculated, preventing information overload.
- **Metric First:** The app defaults to the metric system (kg, cm) for simplicity, as per the Mifflin-St Jeor equation's standard units.
- **Theming:** Supports light and dark modes, respecting user's system preferences for a comfortable viewing experience.

## 4. Key Features

- **BMR Calculation:** Users can calculate their BMR by providing their gender, age, height, and weight.
- **TDEE Calculation:** After calculating BMR, users can select their activity level to determine their TDEE.
- **Responsive Design:** The application is fully responsive and accessible on both desktop and mobile devices.
- **Theme Toggling:** A theme toggler allows users to switch between light and dark modes manually.
