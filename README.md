# BMR & TDEE Calculator Web App

This is a responsive web application designed to help users calculate their Basal Metabolic Rate (BMR) and Total Daily Energy Expenditure (TDEE). Built with Next.js and shadcn/ui, it focuses on simplicity, speed, and ease of use for health and fitness tracking.

## Features

- **BMR Calculation:** Easily calculate your Basal Metabolic Rate by providing your gender, age, height, and weight.
- **TDEE Calculation:** After calculating BMR, you can select your activity level to determine your Total Daily Energy Expenditure.
- **Intuitive User Experience:** The app offers a clean, focused interface, guiding you through the calculations step-by-step.
- **Theme Support:** Supports both light and dark modes, respecting your system preferences for a comfortable viewing experience.
- **Metric System Default:** Calculations default to the metric system (kg, cm) for consistency with the Mifflin-St Jeor equation.

## Getting Started

To run the development server:

```bash
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
